# `<strong>` 태그

#### `<strong>` 태그란?
- `<strong>` 태그는 중요성을 나타내고 싶을 때 사용할 수 있다.
- `<strong>` 시작 태그와 종료 태그 사이에 텍스트를 입력하면 굵은 텍스트로 표기된다.

```html
<body>
    <p>Good <strong>Morning</strong></p>
    <strong>Hey</strong>
</body>
```

<p>Good <strong>Morning</strong></p>
<strong>Hey</strong>
