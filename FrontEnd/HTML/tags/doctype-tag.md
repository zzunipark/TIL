# `<DOCTYPE>` 태그

#### `<DOCTYPE>` 태그란?
- `<DOCTYPE>` 태그는 HTML 태그는 아니지만 브라우저에게 HTML 버전을 알려주는 역할을 한다.

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Hello</title>
    </head>
</html>
```

<small>HTML 4.01은 DOCTYPE을 다르게 선언한다. 다만 HTML5가 주로 사용되므로 생략하겠다.</small>
