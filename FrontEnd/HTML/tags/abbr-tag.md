# `<abbr>` 태그

#### `<abbr>` 태그란?
- `<abbr>` 태그는 축약된 단어 등을 쓸 때 마우스 커서를 글 위에 올려놓으면 원형이 나오게 하고 싶을 때 주로 사용한다.

```html
<body>
    <p><abbr title="HyperText Markup Language">HTML</abbr></p>
</body>
```
