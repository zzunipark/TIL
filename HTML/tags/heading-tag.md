# `<h>` 태그

#### `<h>` 태그란?
- `<h>` 태그는 현재 문서의 제목을 쓸 때 사용할 수 있다.
- `<h>` 태그는 h1 부터 h6 사이를 지정하여 사용할 수 있다.
- 숫자에 따라 글씨의 크기가 달라진다.
- `<p>` 태그와 `<h4>` 태그는 글씨 크기가 같다.

```html
<head>
    <h1>hi</h1>
    <h2>hi</h2>
    <h3>hi</h3>
    <h4>hi</h4>
    <h5>hi</h5>
    <h6>hi</h6>
</head>
```

<h1>hi</h1>
<h2>hi</h2>
<h3>hi</h3>
<h4>hi</h4>
